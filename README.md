# sensual-game-bot
Long poling telegramm bot написанный на Java 17.

## Список используемых технологий ##
Проект написан на java 17, в качестве базы данных была выбрана PostgreSQL.
* ORM - hibernate
* spring MVC 
* spring Data JPA
* spring Boot 2.0
* lombok
* telegram bot api 


## Ознакомиться с проектом ##
Данный проект написан для личного пользования и по сути является тренировочным, но если очень хочется протестировать, то вот инструкция:
1) Запустить проект на heroku: https://sensual-game-bot.herokuapp.com/ (Дождаться 404 ошибки)
2) Написать в телеграмме боту: @first_arkady_bot
